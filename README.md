
# 🎲 Nonebot-Plugin-Lottery (鼠鼠群聊抽奖插件)

一个适用于 Nonebot2 (OneBot V11) 的群聊抽奖与报名系统插件。内置可爱的“鼠鼠”人设，支持**相对/绝对时间定时开奖**、**多项目并发报名**以及**即刻At抽奖**功能。

## ✨ 功能特性

- 🕒 **高自由度定时**：支持类似 `3h后` 的相对时间，以及智能补全的绝对时间（如 `21T18-25`）。
- 📦 **多并发防冲突**：群内可同时发起多个不同的抽奖项目，互不干扰。
- 🙋 **智能报名系统**：只有一个抽奖时一键报名；多个抽奖时支持发送字母（如 `AB`）一键批量报名。
- 🎯 **即刻指定抽奖**：直接 `@` 多名群友，立刻从中随机抽取一名幸运儿。

## 📦 依赖与安装

本插件依赖于 `nonebot_plugin_apscheduler` 提供定时任务支持。

```bash
pip install nonebot-plugin-apscheduler
```

请确保在你的 `pyproject.toml` 或 `bot.py` 中加载了该依赖和本插件。

## 🛠️ 指令说明与使用示例

### 1. 定时抽奖 (创建抽奖)

创建一个在未来指定时间自动开奖的项目。

* **语法**：`/定时抽奖 [项目名称] [时间]`
* **说明**：名称与时间之间**必须有空格**隔开。时间格式支持**相对时间**和**绝对时间**两种。

**【相对时间示例】**
*支持 `h`(小时)、`min`(分钟)、`s`(秒)*

> 用户：`/定时抽奖 肯德基v我50 3h后`
> 用户：`/定时抽奖 随机抽取一个大嘴巴子 30min后`
> 用户：`/定时抽奖 测试项目 99s后`

**【绝对时间示例】**
*使用 `T` 分隔日期和时间。支持智能缺省，漏输的年份默认为今年，漏输的月份默认为当月，漏输的秒数默认为 00。*

* 完整格式：`/定时抽奖 跨年大奖 2026-12-31T23-59-59`
* 缺省年份：`/定时抽奖 劳动节奖 5-1T12-00` (默认今年 5月1日 12:00:00)
* 缺省年月与秒数：`/定时抽奖 今晚夜宵 20T22-30` (默认本月20日 22:30:00)
* 仅指定时间：`/定时抽奖 马上开奖 T18` (默认今天 18:00:00)

### 2. 报名参与

群友发送指令参与当前群内正在进行的定时抽奖。

* **语法**：`/报名` 或 `/报名 [选项字母]`

**【场景 A：当前仅有 1 个抽奖】**

> 用户：`/报名`
> 鼠鼠：报名成功！您已参加【肯德基v我50】的抽奖。

**【场景 B：当前有多个抽奖】**
发送 `/报名` 后，机器人会列出当前所有的项目并分配字母（A、B、C...）。

> 用户：`/报名`
> 鼠鼠：鼠鼠发现有现在多个抽奖项目：
> 第A条：肯德基v我50
> 第B条：拼字比赛
> 请直接回复你想报名的项目对应字母（例如 A，报多个请回复如 AB）
> 用户：`AB` （或者直接使用快捷指令 `/报名 AB`）
> 鼠鼠：成功报名：肯德基v我50, 拼字比赛

### 3. 即时指定抽奖

无需等待，立刻从你选定的人里面随机抽一个。

* **语法**：`/抽奖 @用户1 @用户2 @用户3...`
* **说明**：必须直接 `@` 候选人，可以 `@` 任意数量的群友。

> 用户：`/抽奖 @张三 @李四 @王五`
> 鼠鼠：🎉 开奖啦！恭喜 @张三 (12345678) 成为鼠鼠的幸运儿！

## 📝 注意事项

1. **时间必须在未来**：如果使用绝对时间设置了一个过去的时间，鼠鼠会吐槽并拒绝创建。
2. **防重复报名**：同一个用户对同一个项目多次发送报名，系统会自动拦截，提示已经报名过。
3. **开奖逻辑**：如果抽奖时间到了，但该项目没有任何人报名，抽奖会自动取消并发送提示。
